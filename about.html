Razón del proyecto

Este proyecto fue creado como un ejercicio de programación para practicar habilidades de desarrollo web y lógica de programación. El objetivo es crear un juego interactivo que desafíe a los jugadores a adivinar un número secreto.

El juego utiliza JavaScript para manejar la lógica, HTML para la estructura y CSS para el estilo visual. Es una excelente manera de aprender y aplicar conceptos básicos de programación y desarrollo web.
